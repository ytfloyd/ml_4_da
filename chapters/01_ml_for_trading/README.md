# Chapter 1 — Machine Learning for Trading: From Idea to Execution

The opening chapter of MLAT is mostly conceptual — it frames the ML4T workflow
that every later chapter slots into. We don't fit a model here; we build a
*shared mental model and a one-page picture of our data* that every later
chapter can lean on.

## Goals for our crypto version

1. Confirm the lake matches what we expect (granularities, universe, coverage).
2. Pick the canonical research universe we'll use for early chapters.
3. Sketch the ML4T loop (data → factors → model → strategy → backtest → live)
   as it will look in this repo.
4. Produce a few orientation charts: BTC since 2015, top-50 universe turnover,
   daily-bar coverage by symbol.
